# Auxiliary data files

These data files are used to generate the final data set for the project.

### Directory roadmap

- `scraped.csv` is generated with the `data-scrape.py` file in the `src` directory and was developed by Gian Carlo Diluvi
- `stages_TDF.csv` is a Kaggle data set with years up to 2017. The year 2020 was created in LibreOffice by Gian Carlo Diluvi from information available on https://en.wikipedia.org/wiki/2020_Tour_de_France#Route_and_stages
- `tdf_2020_classifications.csv` was created on LibreOffice by Gian Carlo Diluvi from information available on https://en.wikipedia.org/wiki/2020_Tour_de_France#Classification_leadership_and_minor_prizes
- `tdf-riders.csv` is available online at https://raw.githubusercontent.com/camminady/LeTourDataSet/master/Riders.csv
